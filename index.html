Sélecteurs

Chaque règle CSS commence nécessairement par un sélecteur.

Un sélecteur est un "nom" qui indique à quels éléments d'une page HTML on desire appliquer une règle.

Sélecteurs

  • *
  • element
  • #id
  • .class

Sélecteurs

  • element1 , element2
  • element1 element2
  • element1 > element2
  • element1 + element2
  • element1 ~ element2

Sélecteurs

  • element[attribute]
  • element[attribute=value]
  • element[attribute~=value]
  • element[attribute^=value]
  • element[attribute$=value]

Sélecteurs

  • element:checked
  • element:after
  • element:before
  • element:hover

Le Poids des Sélecteurs CSS

  • 0,0,0,0,0 : Sélecteur universel *, Style du navigateur
  • 0,0,0,0,1 : Sélecteur element
  • 0,0,0,1,0 : Sélecteur classe, Sélecteur d’attribut, Pseudo-classe
  • 0,0,1,0,0 : Sélecteur ID
  • 0,1,0,0,0 : Style en ligne 
    <div style="color: red;">...</li>
  • 1,0,0,0,0 : Bazooka !important

Transitions

Permet une transition douce entre l'ancienne valeur et la nouvelle valeur d'une propriété CSS lorsqu'un événement est déclenché via :

  • Pseudo-classe :hover, :checked
  • JavaScript

Transitions

Pour définir une nouvelle transition animée, il est nécessaire de préciser au minimum :

  • La ou les propriété(s) à animer
  • La durée de l'animation

Transitions

transition-property: width;
transition-duration: 1s;
transition-timing-function: linear;
transition-delay: 2s;

/* Ou bien */
transition: width 1s linear 2s;

Compatibilité

  • Fonctionne parfaitement sur Chrome, Firefox, Safari et Opera
  • IE9 Pour les sélecteurs
  • IE10 Pour les transitions
